# EcoVision: A 3D Pollution Awareness Simulation

## Overview

EcoVision is a 3D environmental awareness project developed using Blender. The project demonstrates the gradual transformation of a clean, green park into a polluted environment, highlighting the negative impact of pollution on nature and public spaces.

Through realistic 3D modeling, animation, lighting transitions, and visual storytelling, the project aims to educate viewers about environmental conservation and encourage responsible waste management practices.

## Features

* Realistic 3D park environment modeling
* Environmental transformation from clean to polluted
* Animated effects including tree deterioration and leaf fall
* Dynamic lighting and atmosphere changes
* Pollution objects such as plastic bottles, paper waste, and cardboard boxes
* Smooth camera movements and scene transitions
* Audio narration for enhanced storytelling

## Technologies Used

* Blender
* 3D Modeling and Texturing
* Animation and Particle Systems
* Lighting and Rendering
* Git & GitHub

## System Architecture

### Modeling Module

Creates and manages park assets such as trees, grass, benches, fences, and pathways.

### Pollution Module

Introduces waste materials and pollution elements that gradually appear throughout the scene.

### Animation Module

Controls environmental changes including tree withering, falling leaves, and pollution spread.

### Material and Lighting Module

Applies realistic textures and lighting conditions to emphasize the contrast between a healthy and polluted environment.

### Camera and Narration Module

Handles camera movement and narration to guide viewers through different stages of environmental change.

## Applications

* Environmental awareness campaigns
* Educational demonstrations
* Social media awareness content
* Sustainability and conservation programs

## Project Outcome

The project successfully demonstrates how pollution can degrade a healthy environment over time and encourages viewers to adopt environmentally responsible behavior to protect natural ecosystems.
